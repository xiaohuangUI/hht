# 项目交接说明

## 一、项目目标

本项目用于“城市交通治理”类比赛展示，强调三件事：

- 展示效果：页面可视化、交互完整、讲解清晰
- 算法能力：预测、检测、策略优化有可解释输出
- 工程可用：可启动、可复现、可分工协作

## 二、推荐分工

- 前端与交互：页面风格、排版、可视化布局、文本讲解
- 算法与模型：时序预测、分类模型、目标检测微调
- 数据与后端：数据清洗、抓取、缓存、导出、接口稳定性
- 汇报与材料：PPT、演示脚本、视频录屏、答辩问答库

## 三、关键运行入口

- 主程序：`app.py`
- 视觉模块核心：`src/vision_lab.py`
- 数据模块核心：`src/traffic_data.py`
- 策略模块核心：`src/traffic_strategy.py`
- 模型实验室：`src/model_lab.py`

## 四、GPU 使用要点

1. 优先使用 `conda run -n pytorch ...` 启动  
2. 在视觉模块中选择“推理设备 = GPU”  
3. 若未检测到 GPU，先执行：

```powershell
conda run -n pytorch python scripts/check_gpu_env.py
```

## 五、演示前检查清单

- 页面无报错（关键模块都能打开）
- YOLO 检测能出框图
- 模型对比图有数据
- 关键图表颜色与字体统一
- 登录功能可用
- 讲解路径固定（3-5分钟版本）
